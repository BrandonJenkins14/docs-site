View content from a Unix System Services (USS) file to your terminal (stdout).

Usage

zowe zos-extended-files view uss-file <file> [options]

Positional Arguments

  • file (string)

    • The name of the USS file you want to print.

Options

  • --binary | -b (boolean)

    • View USS file content in binary mode.

Profile Options

  • --zosmf-profile | --zosmf-p (string)

    • The name of a (zosmf) profile to load for this command execution.

Examples

  • View the contents of the USS file "/a/ibmuser/my_text.txt":

    • zowe zos-extended-files view uss-file "/a/ibmuser/my_text.txt"

  • View the contents of the USS file "/a/ibmuser/MyJavaClass.class" in binary mode:

    • zowe zos-extended-files view uss-file "/a/ibmuser/MyJavaClass.class" --binary
