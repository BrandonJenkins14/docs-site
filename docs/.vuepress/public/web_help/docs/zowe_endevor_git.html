Build changes from git local repository in Endevor

Usage

zowe endevor git <command>

Where <command> is one of the following:

Commands

  • build | b - Build changes against remote git repository's gitBridge main branch, using build area or build subsystem.
  • build-area-list | bal - List build areas in Endevor.
  • build-area-reserve | bar - Reserve build area in Endevor.
  • build-area-unreserve | bau - Unreserve build area in Endevor.
  • request-build-job | rqb - Request to submit build job. [step 1]
  • result-build-job | rsb - Collect result of build job. [step 2]
