Unreserve build area in Endevor.

Usage

zowe endevor git build-area-unreserve <buildAreaId> [options]

Positional Arguments

  • buildAreaId (string)

    • Id of the Endevor build area you would like to unreserve. Name-masking is not allowed.

Options

  • --work-dir | --wd (string)

    • The local working directory of your Git-Endevor mapping that you are building.

      Default value: ./

endevor session definition options

  • --host | --hostname (string)

    • Specifies the base host name.

  • --port | -p (string)

    • Specifies the port number.

  • --protocol | --prot (string)

    • Specifies the protocol used for connecting to CA Endevor SCM Rest API

      Default value: https

      Allowed values: http, https

  • --user | --username (string)

    • Specifies the user name.

  • --password | --pass (string)

    • Specifies the user's password.

  • --reject-unauthorized | --ru (boolean)

    • Specify this option to have the server certificate verified against the list of supplied CAs

  • --base-path | --bp (string)

    • Specifies the base path used for connecting to CA Endevor SCM Rest API

endevor-location definition options

  • --instance | -i (string)

    • Specifies CA Endevor SCM Web Services dataSource name.

Profile Options

  • --endevor-profile | --endevor-p (string)

    • The name of a (endevor) profile to load for this command execution.

  • --endevor-location-profile | --endevor-location-p (string)

    • The name of a (endevor-location) profile to load for this command execution.

Examples

  • Unreserve build area with id "1"This example assumes the endevor profile (used for connection to Endevor web services) is presented:

    • zowe endevor git build-area-unreserve "1" -i ENDEVOR
