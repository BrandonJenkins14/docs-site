View content from a z/OS data set to your terminal (stdout).

Usage

zowe zos-extended-files view data-set <dataSetName> [options]

Positional Arguments

  • dataSetName (string)

    • The name of the data set you want to print.

Options

  • --binary | -b (boolean)

    • View data set content in binary mode.

Profile Options

  • --zosmf-profile | --zosmf-p (string)

    • The name of a (zosmf) profile to load for this command execution.

Examples

  • View the contents of the data set member "ibmuser.cntl(iefbr14)":

    • zowe zos-extended-files view data-set "ibmuser.cntl(iefbr14)"

  • View the contents of the data set member "ibmuser.test.loadlib(main)" in binary mode:

    • zowe zos-extended-files view data-set "ibmuser.test.loadlib(main)" --binary
